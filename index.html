Waarom het HBO bij mij past.

Het HBO past bij mij, omdat ik een nieuwe uitdaging nodig heb.
Ik heb voor het HBO de opleiding MBO bedrijfsadministrateur afgerond en hier ben ik fluitend doorheen gegaan.
Echter zag ik mezelf niet als bedrijfsadministrateur een heel leven werken, daarom had ik voor mijn gevoel iets anders nodig.
Op HBO ICT kan ik mezelf beter gaan ontwikkelen en bouwen aan mijn toekomst.
Vanaf dat ik op de middelbare school zat vond ik computers altijd al interressant.
Hierdoor ben ik mij gaan realiseren dat dit een opleiding voor mij zou kunnen zijn.
Vol vertrouwen ging ik kijken naar een MBO opleidng in de ICT.
Ik ben op een open avond geweest voor MBO ICT en dit werd mij toen sterk afgeraden.
Vervolgens heb ik dit dus niet gedaan en later met spijt ben ik de opleiding bedrijfsadministrateur gaan doen.
Deze heb ik afgerond waardoor ik nu de mogelijkheid heb gekregen om op een hoger niveau ICT te gaan studeren.
Ik heb nog niet veel ervaring met computers, maar ik wil er alles over weten.

Waarom de opleiding bij mijn persoonlijkheid past.

De opleiding past bij mijn persoonlijkheid, omdat mijn kenmerken goed bij deze studie aansluiten.
Een van mijn kenmerken is dat ik een rustig persoon ben.
Dit is handig voor een ICT'er, want ik moet tenslotte heel de dag achter een laptop zitten.
Ik ben nauwkeurig en dit is handig voor het coderen, want hierdoor maak ik weinig fouten.
Zelfstandig werken kan ik goed, ik zorg ervoor dat alles op tijd af komt.
Gedreven, ik ben gedreven om altijd het best mogelijke werk te leveren.
